# Fog Cloud / Туманна хмара

## English Version

### Header

**Name:** Fog Cloud
**Level:** 1st
**School:** Conjuration

### Casting

**Time:** 1 action
**Range:** 120 ft.
**Components:** V, S
**Duration:** Concentration, up to 1 hour

### Description

20-foot-radius sphere of heavily obscured fog.
