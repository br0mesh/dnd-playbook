# Minor Illusion / Мала ілюзія

## English Version

### Header

**Name:** Minor Illusion
**Level:** Cantrip
**School:** Illusion

### Casting

**Time:** 1 action
**Range:** 30 ft.
**Components:** S, M
**Duration:** 1 minute

### Description

Create a sound or image of an object within a 5-foot cube.
