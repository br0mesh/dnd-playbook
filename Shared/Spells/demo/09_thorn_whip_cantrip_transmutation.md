# Thorn Whip / Шипастий батіг

## English Version

### Header

**Name:** Thorn Whip
**Level:** Cantrip
**School:** Transmutation

### Casting

**Time:** 1 action
**Range:** 30 ft.
**Components:** V, S, M
**Duration:** Instantaneous

### Description

Spell attack, 1d6 piercing; pull target 10 ft. closer.
