# Guidance / Настанова

## English Version

### Header

**Name:** Guidance
**Level:** Cantrip
**School:** Divination

### Casting

**Time:** 1 action
**Range:** Touch
**Components:** V, S
**Duration:** Concentration, up to 1 minute

### Description

Target adds 1d4 to one ability check before spell ends.
