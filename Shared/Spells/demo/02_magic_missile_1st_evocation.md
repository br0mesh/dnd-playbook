# Magic Missile / Чарівна стріла

## English Version

### Header

**Name:** Magic Missile
**Level:** 1st
**School:** Evocation

### Casting

**Time:** 1 action
**Range:** 120 ft.
**Components:** V, S
**Duration:** Instantaneous

### Description

Three darts, 1d4+1 force damage each, auto-hit.
