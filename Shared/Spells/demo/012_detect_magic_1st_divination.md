# Detect Magic / Виявлення магії

## English Version

### Header

**Name:** Detect Magic
**Level:** 1st
**School:** Divination

### Casting

**Time:** 1 action
**Range:** Self
**Components:** V, S
**Duration:** Concentration, up to 10 minutes

### Description

Sense magic within 30 ft.; see aura and school.
