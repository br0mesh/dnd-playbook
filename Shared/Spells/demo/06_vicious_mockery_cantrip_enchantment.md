# Vicious Mockery / Безжалісна глузка

## English Version

### Header

**Name:** Vicious Mockery
**Level:** Cantrip
**School:** Enchantment

### Casting

**Time:** 1 action
**Range:** 60 ft.
**Components:** V
**Duration:** Instantaneous

### Description

Wisdom save or 1d4 psychic damage and disadvantage on next attack.
